16.5.3 Syntax colors

The coloring is applied in a background ``thread,'' so you may occasionally see uncolorized text. To change the color scheme, edit the [Colors] section in config.txt.

Python syntax colors:

Keywords
orange
Strings
green
Comments
red
Definitions
blue

Shell colors:
Console output
brown
stdout
blue
stderr
dark green
stdin
black
